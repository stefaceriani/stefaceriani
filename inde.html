What it does

to make Spotify yours

7 Presets

Start from Spotify Default, Midnight Blue, Rose Gold, Forest, Cyberpunk, Monochrome or Light Mode — then make it your own.

13 Color Variables

Control text, subtext, backgrounds, highlights, accent, buttons, sidebar, player bar, cards and notifications.

Live Preview

Colors update instantly as you move the picker. No save needed to see the effect.

Persistent

Your choices are saved via Spicetify.LocalStorage and applied automatically on every launch.

Export / Import

Share your theme as a .json file. Import any theme from the community in one click.

Full Override

Overrides both legacy --spice-* and modern --encore-* Spotify tokens for complete coverage.

Installation

Up and running
in seconds

  1. 1Open Spotify with Spicetify installed
  2. 2Click the Marketplace icon in the top bar
  3. 3Search for ChromaShift
  4. 4Click Install — done!
macOS / Linux 
cp chromashift.js ~/.config/spicetify/Extensions/
spicetify config extensions chromashift.js
spicetify apply
Windows (PowerShell) 
cp chromashift.js "$env:APPDATA\spicetify\Extensions\"
spicetify config extensions chromashift.js
spicetify apply
